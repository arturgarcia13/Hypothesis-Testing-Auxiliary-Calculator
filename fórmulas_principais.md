
# Fórmulas Principais

| Tipo de Teste                                                                      | Hipóteses                                                                                                            | Estatística de Teste                                                                                                                                              | Distribuição sob $H_0$                                                                                       | Região Crítica / Regra de Decisão                                                                                                                                                                                           |
| ---------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Média ($\sigma^2$ conhecida)**                                                   | $H_0: \mu = \mu_0$ <br>$H_1: \mu \neq \mu_0$ / $\mu > \mu_0$ / $\mu < \mu_0$                                         | $Z = \dfrac{\bar{X} - \mu_0}{\sigma / \sqrt{n}}$                                                                                                                  | $N(0,1)$                                                                                                     | Bilateral: $\|Z\| > z_{\alpha/2}$ <br> Unilateral dir.: $Z > z_{\alpha}$ <br> Unilateral esq.: $Z < -z_{\alpha}$                                                                                                            |
| **Média ($\sigma^2$ desconhecida)**                                                | $H_0: \mu = \mu_0$ <br>$H_1: \mu \neq \mu_0$ / $\mu > \mu_0$ / $\mu < \mu_0$                                         | $T = \dfrac{\bar{X} - \mu_0}{S / \sqrt{n}}$                                                                                                                       | $t_{n-1}$                                                                                                    | **Bilateral:** $\|T\| > t_{n-1,\alpha/2}$<br>**Unilateral dir.:**   $T > t_{n-1,\alpha}$  <br>**Unilateral esq.:**  $T < -t_{n-1,\alpha}$  <br><br>                                                                         |
| **Proporção ($p$ conhecida)**                                                      | $H_0: p = p_0$ <br>$H_1: p \neq p_0$ / $p > p_0$ / $p < p_0$                                                         | $Z = \dfrac{\hat{p} - p_0}{\sqrt{p_0(1-p_0)/n}}$                                                                                                                  | $N(0,1)$                                                                                                     | Bilateral: $\|Z\| > z_{\alpha/2}$ <br> Unilateral dir.: $Z > z_{\alpha}$ <br> Unilateral esq.: $Z < -z_{\alpha}$                                                                                                            |
| **Variância ($\sigma^2$ desconhecida)**                                            | $H_0: \sigma^2 = \sigma_0^2$ <br>$H_1: \sigma^2 \neq \sigma_0^2$ / $\sigma^2 > \sigma_0^2$ / $\sigma^2 < \sigma_0^2$ | $\chi^2 = \dfrac{(n-1)S^2}{\sigma_0^2}$                                                                                                                           | $\chi^2_{n-1}$                                                                                               | **Bilateral:** $\chi^2 < \chi^2_{n-1,1-\alpha/2} \quad \text{ou} \quad \chi^2 > \chi^2_{n-1,\alpha/2}$  <br>**Unilateral dir.:**  $\chi^2 > \chi^2_{n-1,,\alpha}$<br>**Unilateral esq.:**  $\chi^2 < \chi^2_{n-1,1-\alpha}$ |
| **Diferença entre médias — Amostras emparelhadas (Teste t pareado)**               | $H_0: \mu_D = \Delta$<br>$H_1: \mu_D \neq \Delta$ / $\mu_D > \Delta$ / $\mu_D < \Delta$                              | $T = \dfrac{\bar{d} - \Delta}{S_d / \sqrt{n}}$ <br><br>onde $\bar{d}$ é a média das diferenças $d_i = x_{i} - y_{i}$ e $S_d$ é o desvio padrão dessas diferenças. | $t_{\,n-1}$                                                                                                  | **Bilateral:** $\|T\| > t_{n-1,\alpha/2}$<br>**Unilateral dir.:**   $T > t_{n-1,\alpha}$  <br>**Unilateral esq.:**  $T < -t_{n-1,\alpha}$                                                                                   |
| **Diferença entre médias ($\sigma_1^2$, $\sigma_2^2$ conhecidas)**                 | $H_0: \mu_1 = \mu_2$ <br>$H_1: \mu_1 \neq \mu_2$ / $\mu_1 > \mu_2$ / $\mu_1 < \mu_2$                                 | $Z = \dfrac{\bar{X}_1 - \bar{X}_2}{\sqrt{\sigma_1^2/n_1 + \sigma_2^2/n_2}}$                                                                                       | $N(0,1)$                                                                                                     | Bilateral: $\|Z\| > z_{\alpha/2}$ <br> Unilateral dir.: $Z > z_{\alpha}$ <br> Unilateral esq.: $Z < -z_{\alpha}$                                                                                                            |
| **Diferença entre médias ($\sigma$ desconhecidas, iguais)**                        | $H_0: \mu_1 = \mu_2$                                                                                                 | $T = \dfrac{\bar{X}_1 - \bar{X}_2}{S_p \sqrt{1/n_1 + 1/n_2}}$ <br> onde $S_p^2 = \dfrac{(n_1-1)S_1^2 + (n_2-1)S_2^2}{n_1 + n_2 - 2}$                              | $t_{n_1+n_2-2}$                                                                                              | Bilateral ou unilateral conforme $H_1$                                                                                                                                                                                      |
| **Diferença entre médias ($\sigma$ desconhecidas e diferentes — Teste t de Welch** | $H_0: \mu_1 = \mu_2$ <br>$H_1: \mu_1 \neq \mu_2$ / $\mu_1 > \mu_2$ / $\mu_1 < \mu_2$                                 | $T = \dfrac{\bar{X}_1 - \bar{X}_2}{\sqrt{W_1 + W_2}}$ <br>com $W_i = \dfrac{S_i^2}{n_i}$                                                                          | $t_v$<br>com $\nu$ ≈ $\dfrac{ \left(W_2 + W_2\right)^2 }{ \dfrac{(W_1)^2}{n_1+1} + \dfrac{(W_2)^2}{n_2+1} }$ | Bilateral: $\|T\| > t_{\alpha/2, \nu}$ <br> Unilateral dir.: $T > t_{\alpha, \nu}$ <br> Unilateral esq.: $T < -t_{\alpha, \nu}$                                                                                             |
| **Diferença entre proporções**                                                     | $H_0: p_1 = p_2$                                                                                                     | $Z = \dfrac{(\hat{p}_1 - \hat{p}_2)}{\sqrt{\hat{p}(1-\hat{p})(1/n_1 + 1/n_2)}}$ <br> com $\hat{p} = \dfrac{f_1 + f_2}{n_1 + n_2}$                                 | $N(0,1)$                                                                                                     | Igual ao caso bilateral/unilateral                                                                                                                                                                                          |
| **Diferença entre variâncias**                                                     | $H_0: \sigma_1^2 = \sigma_2^2$                                                                                       | $F = \dfrac{S_1^2}{S_2^2}$ ou $F = \dfrac{Max(S_1^2, S_2^2)}{Min(S_1^2,S_2^2)}$                                                                                   | $F_{(n_1-1, n_2-1)}$                                                                                         | Bilateral:  $F > F_{v_1, v_2, \alpha/2}$ <br> Unilateral: $F > F_{\alpha}$ ou $F < F_{1-\alpha}$                                                                                                                            |
